Change Log

Alpha 1.11 - TBA

Game

  • Players now select their starting weapon at the lobby screen
  • Asteroids no longer drop weapons after death
  • Gadgets have been added and are bound to right click
  • Ships
    • Health system reworked; Now regenerate health out of combat back to 100%
    • No longer drop weapons after death
  • Weapons
    • Now use power instead of timer based cooldowns
    • Blaster
      • Fire rate increased
    • Photon Cannon
      • Damage increased
      • Spread narrowed
      • Now generates charges the longer it is held
      • The more charges generated the more shots that are fired capping at 3 charges
    • Mass Driver
      • Damage increased
      • Fire rate slightly decreased
    • Particle Beam
      • New weapon added
      • Emits a beam from the ship and delivers a consistant low damage output
      • The longer the beam is maintained the wider and more damage it deals
  • Gadgets
    • PulseWave
      • Creates a wave around the user that pushes enemies away
      • The greater this distance, the more force, as long as enemy is within wave radius
    • Hacking Drone
      • Creates drone that flys towards your mouse position
      • Once a distance has been traveled the drone stops, finds a target, and flys directly towards the target
      • If the drone successfully attaches to a target the drone will disable the ship for a short duration unless damaged
  • Various bug fixes

Website

  • Lobby UI
    • Added ship building menu to customize your ship in the lobby
    • Reworked lobby UI to be more responsive

Alpha 1.1 - 07.13.17

Website

  • Restructured website to accommodate more pages
  • Added mobile support for new layout
  • Fixed issues related to resizing

Engine

  • Fully rebuilt the network code from the ground up
    • Client now only recieves initial information at spawntime
    • Server only sends (trimmed) player information and (trimmed) bullet information to each client per frame
    • Per frame data sent reduced from 10mb/s to 30 kb/s
  • Physics reworked on movement mechanics

Game

  • Added new item - Overdrive
